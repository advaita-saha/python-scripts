# PyTextEditor
## Text Editor Using Python

## Features

- Create new file
- Open existing file
- Various Font support
- Text Alignment
- Bold ,Underline,Italic,text

## Installation

PyTextEditor requires [Python](https://python.org/) v3.9.

Install modules

```sh
pip install PyQt5
```

## Run 

```sh
cd "Python Text Editor"
python main.py
```
